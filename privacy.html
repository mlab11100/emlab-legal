Privacy Policy

Levav Properties - Billing Automation Tool

Last updated: April 6, 2026

This application is an internal tool used solely by Levav Properties LLC. It connects to QuickBooks Online to create invoices on behalf of authorized users.

Data Collection

This application accesses QuickBooks Online data (customer names, invoice data) solely for the purpose of creating and managing invoices. No personal data is collected from end users beyond what is necessary for authentication.

Data Storage

Authentication tokens are stored securely on encrypted servers. Invoice data is stored in a secure PostgreSQL database for audit purposes.

Data Sharing

We do not sell, share, or distribute any data to third parties.
